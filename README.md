🚀 Project Name PCGAY
คำอธิบายสั้นๆ: เว็บไซต์แอปพลิเคชันที่พัฒนาด้วย Next.js สำหรับ E-commerce

📌 Live Demo: https://pcgay.vercel.app/

🛠 Tech Stack
โปรเจกต์นี้สร้างขึ้นด้วยเทคโนโลยีสมัยใหม่เพื่อให้ได้ประสิทธิภาพสูงสุด:

Framework: Next.js (App Router)

Styling: Tailwind CSS / CSS Modules

Deployment: Vercel

Font Optimization: Geist Font (Next/font)

✨ Key Features (ฟีเจอร์เด่น)
Responsive Design: รองรับการใช้งานทุกหน้าจอ (Mobile, Tablet, Desktop)

Fast Loading: เพิ่มประสิทธิภาพการโหลดด้วย Automatic Image & Font Optimization

SEO Friendly: ปรับแต่ง Meta tags เพื่อการทำอันดับบน Search Engine

📖 How to Install (สำหรับนักพัฒนา)
หากต้องการรันโปรเจกต์นี้ในเครื่องของคุณ:

Bash
# 1. Clone โปรเจกต์
git clone https://github.com/YourUsername/pcgay.git

# 2. ติดตั้ง Dependencies
bun install

# 3. รัน Development Server
bun run dev
